Интерактивная игра в формате WebView-приложения: пользователь проходит сюжет, принимает решения и получает разное прохождение.
Сценарии хранятся в JSON и легко расширяются.

Функционал:
1) Ветвящийся сюжет с выбором.
2) Случайные события (ивенты) в перерывах.
3) Вертикальная ориентация (под мобильные устройства).
4) Возможность упаковки в Android/iOS через WebView.

Архитектура и технологии:

Frontend: HTML5, CSS3, JavaScript.
Сценарии: JSON (структура узлов истории, ивентов, концовок).
